<div id="content-box" class="scroll-box content-area" contenteditable="true">### 1. 세금과 공과 계정 복습 [01:12]
- 주요 내용:
- **세금과 공과**는 기업에 귀속되는 다양한 세금(재산세, 자동차세, 지방소득세 등), 공과금, 벌과금, 과태료, 협회비(예: 상공회의소 회비) 등을 비용 처리할 때 사용되는 **판관비** 계정입니다.
- !!주의점!!:
- **자산의 매입 또는 취득에 직접 관련된 세금**은 절대로 **세금과 공과**로 비용 처리하지 않고, 해당 **자산의 취득 원가에 가산**하여 처리해야 합니다.

### 2. 예수금 계정 과목의 이해 [02:59]
- 주요 내용:
- **예수금**은 회사의 돈이 아니라, **타인의 세금이나 공제액을 회사가 잠시 맡아 보관**하고 있는 돈을 처리할 때 사용하는 계정입니다.
- 주로 급여 지급 시 직원의 소득세나 4대 보험료를 공제하는 상황에서 발생합니다.
- **예수금**은 나중에 해당 기관에 납부해야 할 의무가 있는 금액이므로 **유동부채**에 해당하며, 일시적으로 돈을 맡아 보관하는 **임시 가계정**으로 볼 수 있습니다.

### 3. 급여 및 상여금 지급의 회계 처리 [03:25]
- 주요 내용:
- 직원에게 지급하는 **급여**(기본급 및 제수당)는 **급여** 계정(판관비)으로 처리합니다.
- 부정기적으로 지급하는 **상여금**은 **상여금** 계정(판관비)으로 별도 비용 처리합니다.
- 일용직 직원에게 지급하는 급여(일당)는 **잡급** 계정(판관비)으로 처리합니다.
- 급여 지급 시, 직원의 소득세 및 4대 보험료 등 공제액은 **예수금**으로 처리하고, 나머지 차감 지급액만 현금이나 보통 예금 등으로 지급합니다.
- !!주의점!!:
- 시험에서 `직원 급여`와 같은 특정 계정 사용을 지시하지 않는 한, 일반적인 **급여 계정(801번)**을 사용합니다.
- 급여 총액이 비용으로 처리될 금액이며, 상여금 유무를 확인하여 별도 계정으로 처리합니다.

### 4. 원천징수 제도의 개념 및 목적 [08:22]
- 주요 내용:
- **원천징수**는 소득을 지급하는 자(회사)가 소득을 지급받는 자(직원, 용역 제공자 등)의 **소득세를 소득 발생 원천에서 대신 징수하여 납부**하는 제도입니다.
- **원천징수 제도**의 주요 목적은 다음과 같습니다.
1. **세금 포탈 방지 및 효율적인 세금 징수**: 납세자가 직접 세금을 납부하는 것을 회피하려는 경향을 막고, 정부가 안정적으로 세금을 걷을 수 있도록 합니다.
2. **정부 세수 안정화**: 소득세나 법인세처럼 특정 시기에 세금이 집중되는 현상을 완화하고, 소득 발생 시점에 즉시 세금이 징수되어 정부의 재정 운용을 원활하게 합니다.

### 5. 4대 보험료의 회계 처리 [12:15]
- 주요 내용:
- **4대 보험**은 **건강보험**, **고용보험**, **산재보험**, **국민연금**을 지칭합니다.
- 직장 가입자의 4대 보험료는 직원이 절반, 회사가 절반을 부담하는 것이 일반적입니다 (!!예외!!: **산재보험료는 100% 회사가 부담**).
- **종업원 부담분**: 급여에서 공제한 4대 보험료는 **예수금**으로 처리하고, 납부 시 **예수금**을 차변에서 정리하여 현금/보통예금 등으로 지급합니다.
- **회사 부담분**: 회사가 부담하는 4대 보험료는 비용으로 처리하며, 계정 과목은 종류에 따라 다릅니다.
- **국민연금 회사 부담분**: **세금과공과** 계정으로 처리합니다.
- **국민연금을 제외한 나머지 4대 보험 회사 부담분**(건강보험, 고용보험 등): **복리후생비** 계정으로 처리합니다. 이는 사원에게 혜택이 돌아가기 때문입니다.
- !!주의점!!:
- 일반적인 **보험료** 계정(회사가 수혜자)과 **4대 보험료**의 **복리후생비** 계정(사원이 수혜자)을 혼동하지 않도록 주의해야 합니다. 시험에서는 보통 계정 과목을 명시해줍니다.

### 6. 분개 예제 및 실습 해설 (급여, 세금, 4대 보험 납부) [19:15]
- 주요 내용:
- **급여 지급**: (차변) **급여** (총액) / (대변) **예수금** (공제액), **보통예금/현금** (차감 지급액).
- **원천징수 소득세 납부**: (차변) **예수금** (납부액) / (대변) **보통예금/현금**.
- **국민연금 납부**: (차변) **예수금** (종업원 부담분), **세금과공과** (회사 부담분) / (대변) **현금** (총 납부액).
- **건강보험료 납부**: (차변) **예수금** (종업원 부담분), **복리후생비** (회사 부담분) / (대변) **현금** (총 납부액).

### 7. 급여 외 소득 지급 시 예수금 처리 [34:35]
- 주요 내용:
- **예수금**은 급여뿐만 아니라, 회사 입장에서 비용이 발생하거나 지출이 있는 상황에서 **돈을 받는 사람이 소득을 얻어 원천징수 세액이 발생하는 경우**에도 사용됩니다.
- 예시: 개인 김창희씨에게 **자문료** 500만원을 지급하면서 원천징수 세액(16만 5천원)을 공제한 경우.
- (차변) **수수료 비용** 500만원 / (대변) **예수금** 16만 5천원, **보통예금** (나머지 금액).
- !!핵심!!: 회사 입장에서 비용이나 지출이 발생하고, 동시에 **원천징수**라는 표현이 나오면, 언제나 원천징수 세액은 **예수금** 계정으로 처리합니다.

### 8. 원천징수 회계 처리의 시험 출제 및 학습 단계 [39:03]
- 주요 내용:
- **전산회계 2급**: 주로 급여 지급 시 **예수금** 처리 학습.
- **전산회계 1급**: 급여 외 다른 비용(예: 자문료) 지급 시에도 **원천징수**가 발생하며, 이때도 **예수금**을 사용하는 것을 추가 학습.
- **전산세무 2급**: 특정 소득(예: 사업소득)에 대한 원천징수 세율(예: 3.3%)을 직접 계산하는 문제까지 출제될 수 있습니다.
- 회계 학습은 단계적으로 진행되므로, 하위 급수 내용을 정확히 이해하고 있어야 상위 급수 내용을 효율적으로 학습할 수 있습니다.</div>

function formatContent(elementId) {
const box = document.getElementById(elementId);
if (!box) return;

let text = box.innerHTML;
text = text.replace(/^### (.*$)/gim, '<span class="sub-heading">$1</span>');
text = text.replace(/\*\*(.*?)\*\*/g, '<span class="bold-highlight">$1</span>');
text = text.replace(/!!(.*?)!!/g, '<span class="red-highlight">$1</span>');

if (!text.includes('<ul class="custom-list">')) {
const lines = text.split('\n');
let formattedText = '<ul class="custom-list">';
lines.forEach(line => {
const trimmedLine = line.trim();
let liContent = '';
let liClass = '';
if (trimmedLine.startsWith('-')) {
liContent = trimmedLine.substring(1).trim();
} else if (trimmedLine.length > 0) {
liContent = trimmedLine;
}
if (liContent.includes('<span class="sub-heading">')) { liClass = ' is-heading'; }
if (liContent.length > 0) { formattedText += '<li class="' + liClass + '">' + liContent + '</li>'; }
});
formattedText += '</ul>';
text = formattedText;
}
box.innerHTML = text;
}
